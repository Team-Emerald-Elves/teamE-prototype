## Getting Started

This project has two main parts:

- **Frontend** — the client-side application
- **Backend** — the server/API application

You will need to run both the frontend and backend for the full application to work.

---

## Prerequisites

Make sure you have the following installed:

- Node.js
- npm
- pnpm

If you do not have `pnpm` installed, you can install it with:

```bash
npm install -g pnpm
```

---

## Running the Frontend

From the root of the project, navigate into the frontend directory:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## Running the Backend

Open a new terminal window or tab.

From the root of the project, navigate into the backend directory:

```bash
cd backend
```

Install dependencies:

```bash
pnpm install
```

Start the backend development server:

```bash
pnpm run dev
```

---

## Running the Full Project

To run the full application:

1. Open a terminal and start the frontend.
2. Open a second terminal and start the backend.
3. Keep both terminals running while using the application.

## Notes

- The frontend and backend must be running at the same time.
- If one side is not running, parts of the application may not work correctly.
- Check each terminal for errors if the site does not load as expected.
